# Location-App-BE

RESTful API Node Express Mongoose Example

The project builds RESTful APIs using Node.js, Express and Mongoose.

## Manual Installation

Clone the repo:
```
git clone https://github.com/Muhammad-Tahir-S/Location-App-BE.git
```

```
cd Backend-MERN
```

Install the dependencies:
```
npm install
```

Start server:
```
npm start
```
